# Author Royalty System API
Built for the BookLeaf Publishing technical assignment.

### Tech Stack
- **Node.js & Express.js**: Chosen for fast development and efficient handling of JSON REST APIs.
- **In-memory Storage**: Used arrays for data storage as per assignment flexibility to keep the solution clean and simple.

### Assumptions
- All initial sales data dates are in `YYYY-MM-DD` format.
- Withdrawals are deducted from the balance immediately upon the "pending" request.

### Time Spent
- Approximately 3-3.5 hours.
